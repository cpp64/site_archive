Домашние задания: 8 класс

---------------------------------------------------

Примечание: дата д/з указывает день, в который д/з было выдано на уроке. На 1 урок выдается 1 д/з. Если выложено какое-то новое д/з, то оно идет на следующий урок.

Все решения в тетради писать полностью, без полного решения оценка не выставляется.

Д/З на 29 сентября для 8-1,8-3 и на 30 сентября для 8-2,8-4

№1 Расписать таблицу истинности для формулы f = (B → A) ⊕ (A ↓ B)

№2 Расписать таблицу истинности для формулы f = (B ⊕ A) & (A ∨ B)

№3 С++ базовый : составить программу(и записать в тетрадь), которая реализует функцию sg(x):

  1) объявляет целую переменную x

  2) вводит ее с клавиатуры

  3) если x > 0, на экран выводится 1

  4) если x = 0, на экран выводится 0

  5) если x < 0, на экран выводится -1

№4 С++ базовый : составить программу(и записать в тетрадь), которая реализует функцию abs(x):

  1) объявляет целую переменную n

  2) вводит ее с клавиатуры

  3) если n > 0, на экран выводится x

  4) если n = 0, на экран выводится 0

  5) если n < 0, на экран выводится 0-x
