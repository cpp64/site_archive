№3 С++ базовый : составить программу(и записать в тетрадь), которая реализует функцию sg(x):

    1) объявляет целую переменную x

    2) вводит ее с клавиатуры

    3) если x > 0, на экран выводится 1

    4) если x = 0, на экран выводится 0

    5) если x < 0, на экран выводится -1

№4 С++ базовый : составить программу(и записать в тетрадь), которая реализует функцию abs(x):

    1) объявляет целую переменную x

    2) вводит ее с клавиатуры

    3) если x > 0, на экран выводится x

    4) если x = 0, на экран выводится 0

    5) если x < 0, на экран выводится 0-x
