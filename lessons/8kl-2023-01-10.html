1 вариант

№1 Составить программу, которая:

    1) Получает со входа размер массива

    2) Заполняет его так, что ai=i+1 для всех i от 0 до n-1, т.е. 1,2,3,...

    3) Выводит полученный массив на выход

№2 Составить программу, которая:

    1) Получает со входа размер массива и сам массив

    2) Выводит максимальный элемент полученного массива

№3 составить программу, которая:

    1) Получает со входа размер массива и сам массив

    2) Меняет местами минимальный и максимальный элементы полученного массива

    3) Выводит полученный массив на выход

2 вариант

№1 Составить программу, которая:

    1) Получает со входа размер массива

    2) Заполняет его так, что ai=n-i для всех i от 0 до n-1, т.е. n,n-1,n-2,...

    3) Выводит полученный массив на выход

№2 Составить программу, которая:

    1) Получает со входа размер массива и сам массив

    2) Выводит минимальный элемент полученного массива

№3 составить программу, которая:

    1) Получает со входа размер массива и сам массив

    2) Вычитает минимальный элемент полученного массива из максимального (макс. элем. уменьшается на значение минимального)

    3) Выводит полученный массив на выход
